Image Encryption and Decryption
A Python-based application for securing images through encryption and decryption. This project ensures that sensitive images are protected from unauthorized access by transforming them into unreadable formats and reverting them back to their original form when required.

Features
Encrypt Images: Convert images into a secure encrypted format to prevent unauthorized access.
Decrypt Images: Restore encrypted images to their original state for viewing.
User-Friendly GUI: Simple graphical interface built with tkinter for easy interaction.
Password Protection: Add an optional password for extra security.
Clipboard Integration: Copy and use passwords conveniently across different platforms.
